Installation Instructions:

  1. Download the extension file from the website and save it to your computer.
  2. Click the wrench icon on the browser toolbar.
  3. Select Tools > Extensions.
  4. Locate the extension file on your computer and drag the file
  5. onto the Extensions page.
  6. Review the list of permissions in the dialog that appears.
  7. If you would like to proceed, click Install.
